# Weather App
Projeto de um site que informa o clima da região que o úsuario desesja saber utilizando a API da OpenWather.

# 🚀Funcionalidades
- Responsivo
- Informa temperatura máxima, mínima, velocidade do vento e humidade.
- Busca pela região desejada

# 🛠️Tecnologias
- HTML5
- CSS3
- JavaScript
